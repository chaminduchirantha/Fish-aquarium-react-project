# Fish Aquarium Management System

The Fish Aquarium Management System is a full-stack web application designed to manage aquarium products, smart aquarium services, fish orders, accessories, and payment processing. The system also supports delivery management and automated PDF report generation.

This project is built using the **MERN Stack** with **Tailwind CSS** for modern UI styling.

---

## Key Features

###  User Management
- User Registration & Login
- Secure Authentication (JWT-Based)
- Role Support (Customer / Admin)

###  Smart Aquarium System
- Smart Aquarium Device Dashboard and customized the aquarium for the customer.

###  Fish & Accessories Orders
- Fish Product Listing & Ordering
- Aquarium Accessories Ordering
- Shopping Cart & Order Summary
- Order History

###  Payments & Delivery
- Payment Processing
- Delivery Details & Tracking
- Order Status Updates

###  PDF Generator
- Invoice PDF Download
- Order Report PDF
